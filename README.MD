# Projet Recettes MVC

## Installation

1. Cloner le projet ou dézipper l'archive.
2. Importer le fichier `/database/projet_recettes.sql` dans votre gestionnaire de base de données (phpMyAdmin/MySQL).
3. Configurer les accès BDD dans `/src/Core/Db.php` si nécessaire (par défaut : root / sans mot de passe).
4. Lancer le serveur : utilisé laragon
